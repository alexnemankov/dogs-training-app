**Для терминов:**

- **Flashcard flip** — карточка переворачивается при клике (CSS 3D transform). Термин → Определение → Пример.
- **Drag-and-drop сортировка** — перетащи термин к правильному определению.

**Для теории:**

- **Progressive Disclosure** — длинный абзац скрыт за кнопкой «Читать дальше», открывается по клику.
- **Inline Tooltip** — незнакомый термин подчёркнут, при наведении всплывает мини-определение без ухода со страницы.
- **Встроенный micro-quiz** — после каждого тезиса 1 вопрос с немедленной обратной связью (зелёный/красный + объяснение).

**Для практики:**

- **Decision Tree (Дерево решений)** — «Ваша собака делает X → нажмите А или Б» → ветвится к конкретной технике.
- **Scenario Simulator** — описание ситуации, студент выбирает действие, получает анимированную реакцию собаки.

**Для Active Recall:**

- **Spaced Repetition Tracker** — карточки автоматически возвращаются через 1/3/7 дней (можно через localStorage).
- **Confidence Rating** — после ответа студент оценивает уверенность (1–5), это влияет на частоту повторения.
- **Reveal с задержкой** — кнопка «Показать ответ» активируется только через 10 секунд, чтобы форсировать попытку вспомнить.

**Для навигации:**

- **Progress Bar по модулю** — видно, сколько блоков пройдено.
- **Sticky Summary** — плавающая боковая панель с ключевыми тезисами текущего блока.
